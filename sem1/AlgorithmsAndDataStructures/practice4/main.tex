\section*{4.4}
Дан массив неотрицательных чисел длины n. За $\mathcal{O}(n)$ времени найдите количество
его отрезков, сумма его элементов на каждом из которых не превосходит S.

\begin{lstlisting}
    sum = prefix_array(arr)
    j = 0
    ans = 0
    while i < n:
        while (j < n  and sum[j] - sum[i] <= S):
            ans += (j + 1 - i)
            j++
        i++
    print(ans)
\end{lstlisting}
